"# Tic-Tac-Toe-Game" 
This is a web-based Tic-Tac-Toe game built using Python, Flask, HTML, and CSS. The game allows players to play against a friend in multiplayer mode or challenge an AI opponent in single-player mode, providing a fun and interactive experience. Players can enter their names, select the game mode, and see the game board update in real-time. The Flask backend handles game logic, player turns, and AI moves, ensuring smooth and intelligent gameplay. This project demonstrates a combination of frontend and backend development, creating an engaging and user-friendly interface. To run the game, clone the repository, install Flask using pip install flask, run the app.py file, and open the provided URL in a web browser.
